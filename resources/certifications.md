# Certifications for Git, GitHub & DevOps

Relevant certifications to validate your Git/GitHub skills and boost your career. Organized by category.

---

## GitHub Official Certifications

GitHub offers its own certification program. These are the most directly relevant certifications.

| Certification | Level | Cost | Description |
|---------------|-------|------|-------------|
| **GitHub Foundations** | Beginner | $99 | Covers Git basics, GitHub features, collaboration, and project management. Best starting point. |
| **GitHub Actions** | Intermediate | $99 | CI/CD with GitHub Actions, workflow authoring, managing runners, secrets, and environments. |
| **GitHub Advanced Security** | Intermediate | $99 | Code scanning, secret scanning, dependency review, and security policies. |
| **GitHub Administration** | Intermediate | $99 | Managing GitHub at enterprise scale — repos, teams, policies, and compliance. |

**Prep resources:**
- [GitHub Certifications official page](https://resources.github.com/learn/certifications/)
- [Microsoft Learn - GitHub paths](https://learn.microsoft.com/en-us/collections/o1njfe825p602p) (free)
- [GitHub Skills](https://skills.github.com) (free hands-on courses)

## Certification Roadmap

Here's a suggested order for someone going from beginner to professional:

```
Step 1: Learn Git fundamentals
   └── Take: GitHub Foundations cert
       (validates basic Git + GitHub skills)

Step 2: Learn CI/CD
   └── Take: GitHub Actions cert
       (validates automation and pipeline skills)

Step 3: Pick a cloud platform
   ├── Azure path  → AZ-400 (Azure DevOps Engineer)
   ├── AWS path    → AWS DevOps Engineer Professional
   └── GCP path    → Google Cloud DevOps Engineer

Step 4: Specialize
   ├── Security    → GitHub Advanced Security cert
   ├── GitOps      → CKA + ArgoCD/Flux knowledge
   └── IaC         → Terraform Associate
```

## Tips for Certification Prep

1. **Don't rush into paid certs** — Master the fundamentals first. GitHub Foundations is the best starting cert.
2. **Practice > Theory** — Hands-on experience with real repos matters more than memorizing docs.
3. **Use free resources first** — Microsoft Learn, GitHub Skills, and Coursera (audit mode) cover most prep material for free.
4. **Build projects** — Certifications test practical knowledge. Build the projects from `practice-projects.md` before taking exams.
5. **Check for discounts** — Many cert providers offer discounts for students, during events (Black Friday, re:Invent), or through employer learning programs.
6. **Verify before purchasing** — Certification programs change. Always check the official website for current pricing, availability, and exam content before registering.

---

> **Recommendation:** Start with **GitHub Foundations** — it's affordable, directly relevant, recognized by employers, and the prep material is mostly free via Microsoft Learn and GitHub Skills.

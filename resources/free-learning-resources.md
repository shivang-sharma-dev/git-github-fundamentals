# Free Learning Resources for Git & GitHub

A curated list of the best free resources to learn Git and GitHub — from absolute beginner to advanced.

---

## Official Documentation

| Resource | Description |
|----------|-------------|
| [Git Official Docs](https://git-scm.com/doc) | The authoritative reference for all Git commands and concepts |
| [GitHub Docs](https://docs.github.com) | Official GitHub documentation covering every feature |
| [Git Reference Manual](https://git-scm.com/docs) | Detailed command-by-command reference |
| [GitHub Skills](https://skills.github.com) | Interactive courses by GitHub to learn Git and GitHub hands-on |

---

## Free Courses & Tutorials

### Beginner

| Resource | Platform | Description |
|----------|----------|-------------|
| [Introduction to Git and GitHub](https://www.coursera.org/learn/introduction-git-github) | Coursera (Google) | Part of Google IT Automation with Python. Covers Git basics, branching, and GitHub collaboration |
| [Git & GitHub Crash Course](https://www.youtube.com/watch?v=RGOj5yH7evk) | YouTube (freeCodeCamp) | Full beginner-friendly crash course (~1 hr) |
| [Version Control with Git](https://www.udacity.com/course/version-control-with-git--ud123) | Udacity | Free course covering Git fundamentals |
| [Learn Git Branching](https://learngitbranching.js.org) | Interactive Web | Visual and interactive way to learn Git branching — highly recommended |
| [GitHub Skills Courses](https://skills.github.com) | GitHub | Hands-on courses: Introduction to GitHub, Communicate using Markdown, GitHub Pages, etc. |

### Intermediate

| Resource | Platform | Description |
|----------|----------|-------------|
| [Git for Professionals](https://www.youtube.com/watch?v=Uszj_k0DGsg) | YouTube (freeCodeCamp) | Covers advanced workflows, rebase, stash, reflog |
| [How to Contribute to Open Source](https://opensource.guide/how-to-contribute/) | GitHub Open Source Guide | Step-by-step guide to making your first open source contribution |
| [The Odin Project - Git Basics](https://www.theodinproject.com/paths/foundations/courses/foundations#git-basics) | The Odin Project | Practical Git lessons as part of a full web dev curriculum |
| [Atlassian Git Tutorials](https://www.atlassian.com/git/tutorials) | Atlassian | Well-written tutorials on branching, merging, workflows, and advanced topics |

### Advanced

| Resource | Platform | Description |
|----------|----------|-------------|
| [Git Internals (Pro Git Book - Ch. 10)](https://git-scm.com/book/en/v2/Git-Internals-Plumbing-and-Porcelain) | Git SCM | Deep dive into Git's internal object model, packfiles, and plumbing commands |
| [Advanced Git (Pluralsight - free trial)](https://www.pluralsight.com/courses/advanced-git) | Pluralsight | Advanced topics: interactive rebase, cherry-pick, bisect, submodules |
| [Git from the Bottom Up](https://jwiegley.github.io/git-from-the-bottom-up/) | Free Online Book | Teaches Git by starting from its internal data structures |
